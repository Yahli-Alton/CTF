Welcome to Yvm, the questionable JVM.

Yvm executes (some) JVM bytecode. It supports:

  • ints, chars, (multi-dimensional) char[]s
  • static methods
  • a native print method
Put differently: The Yvm does not support OO, anything from the Java Standard Library, long, and all the other things you like about Java. Admire the big match statement in src/interpreter.ml to see which opcodes are supported.
